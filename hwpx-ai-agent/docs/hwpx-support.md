# HWPX Support

## 지원

- ZIP 기반 HWPX 패키지 열기
- section/body XML 탐색
- 문단 텍스트 추출
- 표/행/셀 추출
- 셀 숫자 Decimal 정규화
- 정확한 텍스트 검색과 교체
- 표 셀 값 수정
- 생성된 초안 문단을 본문 끝에 추가
- 새 HWPX 파일 생성
- XML 파싱 가능 여부 검증
- 수정 전후 XML diff

## 제한

- 이미지, 도형, 차트, 수식은 MVP에서 수정하지 않는다.
- 복잡한 HWPX 스타일 병합은 원본 run 구조를 가능한 한 보존하는 수준으로 처리한다.
- 완전한 새 HWPX 생성기는 아직 아니며, MVP의 작성 기능은 기존 HWPX 작업 사본에 초안 문단을 추가하는 방식이다.
- 공식 HWPX XSD 검증은 확장 지점만 마련했다.
